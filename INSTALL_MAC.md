# Установка AURA на macOS

## Если macOS блокирует запуск приложения

При первом запуске macOS может показать сообщение:
- «Приложение AURA.app нельзя открыть, так как оно от неизвестного разработчика»
- или «Файл повреждён, его нужно переместить в корзину»

Это ограничение Gatekeeper. Для запуска приложения используйте один из способов ниже.

### Способ 1: Правая кнопка мыши

1. Щёлкните **правой кнопкой мыши** (или Control + клик) по AURA.app
2. Выберите **«Открыть»**
3. В появившемся диалоге нажмите **«Открыть»**

После этого приложение добавится в список разрешённых и будет запускаться обычным двойным кликом.

### Способ 2: Терминал

1. Откройте приложение **Терминал**
2. Выполните команду (подставьте свой путь к AURA.app):

```
xattr -cr /путь/к/AURA.app
```

Например, если приложение в папке «Загрузки»:

```
xattr -cr ~/Downloads/AURA.app
```

Команда снимает атрибуты карантина с приложения. После этого запустите AURA.app обычным способом.
