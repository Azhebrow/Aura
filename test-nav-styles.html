Navigation Redesign - Тестирование

Это тестовая страница для проверки стилей новой навигации.

Откройте DevTools (F12) и изменяйте размер окна, чтобы видеть адаптивные стили:

  • Полный размер (>1100px): обе панели видны с полной шириной
  • Планшет (640-1100px): панели адаптируются
  • Мобиль (≤640px): нижняя панель становится плавающим dock'ом
  • Очень узкий (≤480px): все элементы сжимаются

Основные функции:

  • ✓ Backdrop-filter с размытием 8px
  • ✓ Полупрозрачный фон (85% opacity)
  • ✓ Плавные переходы (0.3s ease)
  • ✓ Fallback для браузеров без backdrop-filter
  • ✓ iPhone-style dock на мобилях (90% ширины, 12px отступ снизу)
  • ✓ Нативная граница сверху/снизу
  • ✓ Правильные тени для elevation
