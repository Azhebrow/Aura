# Звуковая система AURA

## Описание

Эта папка содержит все звуковые файлы для аудиоинтерфейса приложения AURA. Все звуки в формате **WAV**. Все звуки находятся в одной папке для простоты управления.

## Структура звуков

### Формат файлов
- **WAV** - все звуки интерфейса

### Типы звуков

1. **UI взаимодействия**
   - `small-buttons.wav` - звук кнопок
   - `success.wav` - успешное действие
   - `error.wav` - ошибка
   - `text-area.wav` - фокус на текстовые поля

2. **Таймер**
   - `timer-start.wav` - запуск таймера
   - `finished-timer.wav` - завершение таймера
   - `cansel-timer.wav` - отмена таймера

3. **Данные**
   - `task-completed.wav` - завершение задачи

4. **UI навигация**
   - `select.wav` - выбор элементов, radio buttons
   - `toggle_on.wav` - включение состояния
   - `toggle_off.wav` - выключение состояния
   - `caution.wav` - предупреждения, alert

5. **Рандомные звуки для кнопок**
   - `type_01.wav` - `type_05.wav` - случайный выбор для кнопок

6. **Рандомные звуки для ритуалов**
   - `tap_01.wav` - `tap_05.wav` - случайный выбор для ритуалов

## Добавление новых звуков

### Шаг 1: Подготовка файла
1. Убедитесь, что файл в формате WAV
2. Рекомендуемая длительность: 0.1-0.5 секунды для кликов, до 2 секунд для событий
3. Рекомендуемое качество: 16-bit, 44.1 kHz или выше
4. Формат имени: `kebab-case.wav` (например, `button-click.wav`)

### Шаг 2: Размещение файла
Поместите файл **напрямую в эту папку** (`src/system/audio/sounds/`). Все звуки находятся в одной папке, без подпапок.

### Шаг 3: Настройка конфигурации
Откройте `src/system/audio/soundConfig.js` и добавьте маппинг:

```javascript
// Для события EventBus
eventBusSounds: {
  'ritualCompleted': 'ritual-completed',
  'yourNewEvent': 'your-new-sound'  // Добавьте здесь
}

// Или для кнопок
buttonSounds: {
  'default': 'click',
  'your-button-class': 'your-new-sound'  // Добавьте здесь
}
```

### Шаг 4: Использование в коде

```javascript
import { audioSystem } from './system/services/index.js';

// Автоматически через EventBus (если настроено в soundConfig.js)
eventBus.emit('yourNewEvent', {...});

// Или напрямую
audioSystem.play('your-new-sound');

// Или привязка к кнопке
audioSystem.attachToButton(buttonElement, 'your-new-sound');
```

## Формат именования

Используйте `kebab-case` для имен файлов:
- ✅ `button-click.wav`
- ✅ `ritual-completed.wav`
- ✅ `timer-start.wav`
- ❌ `buttonClick.wav` (camelCase)
- ❌ `button_click.wav` (snake_case)
- ❌ `Button Click.wav` (с пробелами)

## Рекомендации

1. **Громкость**: Звуки должны быть нормализованы по громкости
2. **Длительность**: Короткие звуки (0.1-0.5 сек) для кликов, немного длиннее для событий
3. **Качество**: Используйте качественные звуки без артефактов
4. **Тестирование**: Проверьте звуки на разных уровнях громкости системы

## Текущие звуки

После добавления ваших файлов, список будет автоматически обновлен. Убедитесь, что все необходимые звуки присутствуют в папке.
