## Использование:

Все тени генерируются автоматически в зависимости от темы, акцента и уровня ('subtle', 'moderate', 'strong').

```css
/* Использование */
.card {
  box-shadow: var(--shadow-card-level);  /* автоматически subtle */
}

.section {
  box-shadow: var(--shadow-section-level);  /* автоматически subtle */
}

.sticky-header {
  box-shadow: var(--shadow-sticky-header);  /* адаптируется к теме */
}
```

Эффекты теперь **мягкие, элегантные и не отвлекают**.
